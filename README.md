# ひび割れ一括解析・アノテーションツール

建築物の外壁写真を Google Gemini API で解析し、ひび割れや欠損・剥離箇所を自動検出して YOLO セグメンテーション形式のラベルを生成する Streamlit アプリです。

## 機能

- 複数画像の一括アップロード・順次処理
- Gemini API による劣化箇所の自動検出（赤色描画）
- ポリゴン描画による誤検知エリアの除外
- 除外後のリアルタイムプレビュー
- YOLO セグメンテーション形式（`.txt`）への変換
- 確認用可視化画像と合わせた ZIP 一括ダウンロード

## 検出種類

| 種類 | 内容 |
|------|------|
| ひび割れ | 建材表面のひび割れを赤線で描画 |
| 欠損・剥離 | アスペクト比 0.5〜2.0 の欠損・剥離部を赤色で塗りつぶし |
| その他 | ひび割れ検出（汎用プロンプト） |

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` を開きます。

## 使い方

### 1. 初期設定（サイドバー）

| 項目 | 説明 |
|------|------|
| Gemini API Key | Google AI Studio で取得した API キー |
| モデル | `gemini-3-pro-image-preview` または `gemini-3.1-flash-image-preview` |
| 検出種類 | ひび割れ / 欠損・剥離 / その他 |
| 最小ポリゴン面積(px) | この面積以下のノイズ輪郭を除外（デフォルト: 10px） |

### 2. 画像のアップロード

対応形式: JPG, PNG, JPEG（複数選択可）

### 3. 解析

「この画像を解析」ボタンをクリックすると Gemini API が画像を解析し、右側に検出結果（赤色描画）を表示します。

### 4. 誤検知エリアの除外（任意）

右側のキャンバスで誤検知エリアを多角形で囲みます。

| 操作 | 動作 |
|------|------|
| クリック | 頂点を追加 |
| ダブルクリック | 直前の頂点を削除 |
| 右クリック | 多角形を確定 |

確定後、除外処理を適用したプレビューがキャンバス下に自動表示されます。

### 5. 確定・保存

「この画像の結果を確定して保存」をクリックすると、誤検知除外済みの画像から YOLO ラベルを生成してセッションに保存します。

サイドバーの「前へ / 次へ」で次の画像に移動し、同じ手順を繰り返します。

### 6. 一括ダウンロード

すべての画像を確定後、「全ての確定済みデータを ZIP でダウンロード」から出力します。

ZIP の構成:

```
yolo_dataset_all.zip
├── labels/
│   ├── image1.txt
│   └── image2.txt
└── visualized/
    ├── image1.jpg
    └── image2.jpg
```

- `labels/`: YOLO セグメンテーション形式のアノテーション（クラス ID `0` 固定）
- `visualized/`: 検出輪郭を描画した確認用画像

## ファイル構成

```
.
├── app.py          # Streamlit UI
├── logic.py        # Gemini API 呼び出し・YOLO 変換処理
└── requirements.txt
```

| numpy | 配列演算 |
| Pillow | 画像読み込み・変換 |
